                请按顺序执行以下命令
════════════════════════════════════════════════════════════

步骤 1: 确认最新 APK 是否已生成
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

cd "/Users/apple/Downloads/hiddify-app-main 2"
ls -lh build/app/outputs/flutter-apk/app-arm64-v8a-debug.apk

查看文件时间，应该是今天 21:50 之后。

════════════════════════════════════════════════════════════

步骤 2: 卸载手机上的旧版本
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

adb uninstall app.hiddify.com

════════════════════════════════════════════════════════════

步骤 3: 安装最新版本
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

adb install build/app/outputs/flutter-apk/app-arm64-v8a-debug.apk

════════════════════════════════════════════════════════════

步骤 4: 查看日志（自动化脚本）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

./完整测试步骤.sh

这个脚本会：
- 自动检查手机连接
- 自动检查应用安装
- 自动清除旧日志
- 自动监控新日志

然后在手机上点击连接按钮，就能看到日志了。

════════════════════════════════════════════════════════════

如果还是没有日志，运行：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

adb logcat

会显示所有日志（大量滚动），然后在手机上点击连接，
用眼睛找包含 "permission" 或 "denied" 的行。
